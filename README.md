# 🇺🇸 Build a Survey Form (English)

**Improvement Survey Form** – A responsive and customizable survey form built for the freeCodeCamp certification project, featuring a modern design and user-friendly interface.

## 📋 About the Project
This project is a survey form designed to collect user feedback, suggestions, or issues. It includes fields for name, email, age, referral source, feedback type, topic selection, and a detailed message area. The design goes beyond the basic requirements, adding a personalized touch with custom CSS animations, hover effects, and a carefully chosen color palette.

[**freeCodeCamp – Build a Survey Form**](https://www.freecodecamp.org/learn/2022/responsive-web-design/build-a-survey-form-project/build-a-survey-form)

## 🚀 Features
- **Fully responsive** – adapts to different screen sizes (mobile, tablet, desktop)
- **Custom styled form elements** – radio buttons, checkboxes, dropdown, and text inputs with smooth transitions
- **Animated interactions** – hover and focus effects for better user experience
- **Background image overlay** – semi‑transparent overlay for better text readability
- **Accessible markup** – proper use of labels, fieldsets, and legends
- **HTML5 validation** – built‑in validation for required fields, email format, and number range

## 🛠️ Technologies & Concepts Used
- **HTML5** – semantic structure, form elements, input types
- **CSS3** – Flexbox, CSS Custom Properties (variables), media queries, transitions
- **Google Fonts** – Barlow font family
- **Custom form controls** – pure CSS checkboxes and select dropdown styling
- **Background techniques** – linear‑gradient overlay on background image

## 🧠 What I Learned
- Reinforced understanding of HTML form structure and accessibility best practices.
- Practiced CSS theming with custom properties for consistent colors and easier maintenance.
- Implemented custom‑styled checkboxes and radio buttons without JavaScript.
- Used background overlays to improve text contrast over images.
- Applied responsive design principles to ensure usability on all devices.

# 🇧🇷 Construa um Formulário de Pesquisa (Português)

**Improvement Survey Form** – Um formulário de pesquisa responsivo e personalizável, desenvolvido para o projeto de certificação do freeCodeCamp, com design moderno e interface amigável.

## 📋 Sobre o Projeto
Este projeto é um formulário de pesquisa criado para coletar feedback, sugestões ou problemas dos usuários. Inclui campos para nome, e‑mail, idade, fonte de referência, tipo de feedback, seleção de tópicos e uma área para mensagem detalhada. O design vai além dos requisitos básicos, adicionando um toque pessoal com animações CSS personalizadas, efeitos de hover e uma paleta de cores cuidadosamente escolhida.

[**freeCodeCamp – Construa um Formulário de Pesquisa**](https://www.freecodecamp.org/portuguese/learn/2022/responsive-web-design/build-a-survey-form-project/build-a-survey-form)

## 🚀 Funcionalidades
- **Totalmente responsivo** – adapta‑se a diferentes tamanhos de tela (celular, tablet, desktop)
- **Elementos de formulário personalizados** – botões de rádio, caixas de seleção, menu suspenso e campos de texto com transições suaves
- **Interações animadas** – efeitos de hover e foco para melhor experiência do usuário
- **Sobreposição de imagem de fundo** – camada semitransparente para melhor legibilidade do texto
- **Marcação acessível** – uso adequado de labels, fieldsets e legends
- **Validação HTML5** – validação nativa para campos obrigatórios, formato de e‑mail e faixa numérica

## 🛠️ Tecnologias & Conceitos Utilizados
- **HTML5** – estrutura semântica, elementos de formulário, tipos de input
- **CSS3** – Flexbox, propriedades personalizadas (variáveis CSS), media queries, transições
- **Google Fonts** – família de fontes Barlow
- **Controles de formulário personalizados** – caixas de seleção e menu suspenso estilizados apenas com CSS
- **Técnicas de fundo** – sobreposição com gradiente linear sobre imagem de fundo

## 🧠 Aprendizados
- Reforcei o entendimento da estrutura de formulários HTML e práticas de acessibilidade.
- Pratiquei a criação de temas com propriedades personalizadas CSS para cores consistentes e manutenção mais fácil.
- Implementei caixas de seleção e botões de rádio personalizados sem JavaScript.
- Usei sobreposições de fundo para melhorar o contraste do texto sobre imagens.
- Apliquei princípios de design responsivo para garantir usabilidade em todos os dispositivos.
